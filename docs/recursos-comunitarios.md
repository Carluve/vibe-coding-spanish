## Canales de Discord recomendados

El servidor de Discord de Cloudflare Developers es muy activo. Algunos canales útiles:

- `#general` — preguntas generales
- `#workers` — dudas sobre Cloudflare Workers y Wrangler
- `#workers-ai` — IA, Agents SDK, MCP
- `#pages` — Cloudflare Pages y frameworks
- `#d1` — base de datos D1
- `#r2` — almacenamiento R2
- `#durable-objects` — Durable Objects y estado persistente
- `#zero-trust` — Cloudflare One, Access, Tunnel

## Mantenerse actualizado

### Blog — Etiquetas recomendadas

- [blog.cloudflare.com/tag/workers](https://blog.cloudflare.com/tag/workers/) — novedades de Workers
- [blog.cloudflare.com/tag/ai](https://blog.cloudflare.com/tag/ai/) — IA y Agents

### Innovation Weeks

Cloudflare organiza semanas temáticas de anuncios varias veces al año:
- **Birthday Week** (septiembre) — anuncios generales
- **Security Week** (marzo) — seguridad y Zero Trust
- **Developer Week** (abril) — plataforma de desarrollo
- **Privacy Week** (enero) — privacidad y compliance
